# Övningar för Applied Script, 20 yh Frans Schartaus Handelsinstitut

Detta repository innehåller övningar för kursen Applied Script som genomfördes under vintern 2025/26.


## Översikt

|    Projekt   | Beskrivning                                  |
|--------------|----------------------------------------------|
| **o1**  | Informationsinsamlingsscript i Linux-miljö  |
| **o2**  | MD5-hasher och bruteforce-attack med Hashcat|
| **o3**  | AV/EDR-test med EICAR-testfil och antivirusrespons|


### Övning 1: Informationsinsamlingsscript (RECON)
I denna övning skapandes ett bash-script som automatiskt samlar in systeminformation från
en Linux-miljö. Scriptet ger en grundläggande överblick systemdata.

### Övning 2: MD5-hasher och Hashcat Brute-Force

Den andra övningen fokuserade på att generera MD5-hashar från slumpmässiga lösenord och sedan
använda Hashcat för att testa hash-värdena kan knäckas. 

### Övning 3: AV/EDR Test med EICAR-testfil

Den tredje övningen gick ut på att testa hur antivirus- eller EDR-system reagerar på ett
känt testvirus som EICAR-signaturen.



## Mål

- **Övning 1:** Automatisera insamling av systeminformation för att få en översikt över ett Linux-system.
- **Övning 2:** Förstå hur hashfunktioner(MD5) kan knäckas med hjälp av brute-force-verktyg(Hashcat)
och visa begränsningar med MD5.
- **Övning 3:** Verifiera hur ett antivirus- eller EDR reagerar på en känd testvirusfil 
och utvärdera effektiviteten av det skyddssystemet.
